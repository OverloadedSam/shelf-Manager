About

Shelf Manager is a simple web app for managing books and creating a list of books that you currently have. Shelf Manager is great tool for adding, removing and managing books from your current stock/book shelf. Everybody can use this app Whether he/she is a Student, Book seller or a Librarian. With Shelf Manager you can save your books permanently in the local storage of your browser for free without any problem. This app focuses on simplicity. The user interface is so friendly and minimalistic that everybody will love it regardless of any age group.
